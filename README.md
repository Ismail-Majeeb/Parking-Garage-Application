# Parking-Garage-Application
This simple application manages parking spaces for a configurable maximum number of vehicles. Each parking space (slot) defined with a dimension(width and depth). This application is designed to improve the parking experience for customers.

The **Purpose** of this Software is to:
- **Track and manage parking garage occupancy.**
- **Allow consumers to locate and book available parking spaces**.

This will save time spent looking for open spots and, as a result, reduce traffic congestion that may occur inside the garage or on the streets as drivers look for parking spaces.
